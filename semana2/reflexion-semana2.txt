es importante para un desarrollador saber usar la terminal porque permite trabajar de manera mas rápida y organizada debtro de un proyecto. la terminal ayuda a crear carpetas, mover archivos y ejecutar comandos importantes. también facilita el uso de las herramientas como Git y GitHub para guardar cambios y trabajar en equipo.
